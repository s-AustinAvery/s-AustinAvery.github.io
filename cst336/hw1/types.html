Overview Complexity Types Design Techniques

Common Categories of Algorithms

  • Sorting: Merge Sort [O(n log n)], Quick Sort [average O(n log n)].
  • Searching: Binary search [O(log n)] when used with sorted data.
  • Graph: Breadth First Search, Depth First Search, Dijkstra, Kruskal.
  • Numeric: Euclid GCD, fast exponentiation.
BFS and DFS

Breadth First Search explores a data structure level by level, where as Depth First Search first explores a route as deeply as it can go.

What is Complexity?

Complexity offers a way to compare algorithms without regard to the machine they may be run on. With an input size of n, we can compare how an algorithm would scale between constant O(1), logarithmic O(log n), linear O(n), and quadratic O(nÂ²) among others.

Worst case vs average case

Worst case uses the hardest possible scenario, while average case averages over a distribution.
